<!-- GETTING STARTED -->
## Getting Started

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/ntijoh-my-lovell/webbshop-te4-angular.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Running the application
   ```sh
   ng serve
   ```
